# n8n_feishu-ocr-score-api
1. 工作流概述：实现了对读后续写的题目&amp;作答图片ocr识别、信息集成评分，使得数据获取、AI处理与结果反馈的工作流自动化。 2.技术选型：最终使用飞书社区节点、飞书应用，在多维表格上传需要处理的文件，写回识别内容与评分结果。AI处理上，调用了火山引擎视觉识别、硅基流动评分的大模型api。 3.工程细节： 3.1 按照模块任务，分解多个子节点，例如ocr识别时，包括文件下载、图像转码和文本识别。 3.2 贯穿工作流的是，使用Code节点进行数据清洗，进行节点间出入参的格式调整转换，结合Merge节点，实现必要的合并，简单数据清晰可使用Aggregate、Filter节点提取、筛选。
